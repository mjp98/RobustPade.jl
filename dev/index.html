RobustPade.robustpade — Method

robustpade( coeffs::AbstractVector{T}, m::Integer, n::Integer; tol::Real=eps(float(real(T))) ) where T computes the (m,n) Pade approximant to a function with Taylor coefficients 'coeffs' using SVD following Parchon et al. SIAM Review.

Adapted from the Chebfun implementation at https://github.com/chebfun/chebfun/blob/master/padeapprox.m (modified BSD license)
